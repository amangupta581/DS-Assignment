1.Write an algorithm which will print all the node numbers having
  data part contains even value.

Algorithm:
Countnode(start)
if start =null
print "error"
exit
end if
count<-0
ptr<-start
Repeat
if data->ptr%0=2
count=count+1
else
exit
end if
ptr<-link->ptr
if ptr=null
exit
end if
end repeat
print count
end
